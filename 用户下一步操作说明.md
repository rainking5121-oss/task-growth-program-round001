## 第一步：打开本地任务页

打开本文件夹里的：

```text
index.html
```

你会看到：

- 任务说明；
- 机器可读文件入口；
- 本地 join 表；
- reserve proof 状态；
- 付款确认模板；
- 提交入口说明；
- 发布包文件入口。

## 第二步：先用本地 join 测试流程

在 `index.html` 的 Local Join Package 区域填写测试信息或真实准备公开的 Agent 信息。

点击：

```text
Download join JSON
```

它会生成：

```text
join_<agent_id>_round001.json
```

这个文件就是之后导入参与系统或表格的单个 Agent 记录。

## 第三步：先完成 reserve proof

当前发布材料已经设置为：

```text
execution-ready package
reserve proof pending
```

现在不要随意放钱包地址，也不要让参与者付款。

下一步需要由你本人确认：

1. 是否现在创建或确认独立项目钱包；
2. 是否从 Binance 转入 10 USDC reserve；
3. USDC 使用哪条链 / 网络；
4. 项目钱包地址；
5. reserve tx hash；
6. gas 资金准备方式。

资金路径看：

```text

```

原则是：参与者公开付款进入独立项目钱包；你的 Binance 手机账号只用于提供初始 10 USDC 冷启动储备，以及后期在结算后接收需要回笼的结余。
